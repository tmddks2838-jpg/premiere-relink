# Premiere Relink — 프리미어 오프라인 미디어 자동 재연결

> 프리미어 프로를 열었는데 클립들이 빨갛게 **"미디어 오프라인"** 으로 떠 있을 때,
> 파일을 하나하나 찾아서 다시 연결하지 않아도 **알아서 다 연결해주는** 무료 맥 앱입니다.

이런 경우에 쓰면 됩니다:

- 외장하드 이름이 바뀌어서 영상이 전부 오프라인이 됐을 때
- 영상 폴더를 다른 곳으로 옮겼을 때
- 다른 맥에서 만든 프로젝트를 가져왔을 때

**원본은 절대 건드리지 않습니다.** 결과는 항상 새 파일(`_relinked.prproj`)로 따로 저장되고, 작업 전에 백업도 자동으로 만들어요. 잘못돼도 원본은 그대로입니다.

- 인텔 맥 · M칩 맥 **모두 지원**
- 현재 버전: **v0.1** (베타 — 버그 제보 환영!)

## 🔧 설치 방법 — 처음 한 번만, 딱 3단계

> 💬 **시작 전에 하나만 알아두세요.**
> 이 앱은 앱스토어에서 받은 게 아니라서, 그냥 열면 맥이
> "확인할 수 없는 앱"이라며 **막아버립니다.** 바이러스라서가 아니라,
> 제작자가 애플에 연회비(약 13만원)를 안 내서 뜨는 경고예요.
> 그래서 아래 2단계에서 **터미널에 한 줄만 붙여넣으면** 이 잠금이 풀립니다.
> 한 번만 하면 그 다음부터는 평생 더블클릭으로 열려요.

### 1단계 — 앱을 "응용 프로그램" 폴더에 넣기

1. 다운로드한 `Premiere Relink.zip`을 더블클릭하면 압축이 풀립니다.
2. 풀린 폴더 안에 **Premiere Relink** 라는 앱이 있어요.
3. 이 앱을 **응용 프로그램(Applications)** 폴더로 끌어다 놓으세요.
   - 응용 프로그램 폴더 찾는 법: 바탕화면에서 `Command + Shift + A` 를 누르면 바로 열립니다.

### 2단계 — 터미널에 한 줄 붙여넣기 (잠금 풀기)

"터미널"이라고 하면 어려워 보이지만, **복사 → 붙여넣기 → Enter** 가 전부예요.

1. 키보드에서 `Command + Space` 를 눌러요. (화면에 검색창이 뜹니다)
2. `터미널` 이라고 치고 Enter. (까만 창이 하나 열립니다 — 이게 터미널이에요)
3. 아래 한 줄을 **통째로 복사**해서 터미널에 붙여넣고 Enter:

```
xattr -cr "/Applications/Premiere Relink.app"
```

4. 아무 메시지 없이 조용히 넘어가면 **성공**입니다. (원래 아무것도 안 떠요)

> 🙋 "이 명령어 뭐 하는 건데요?"
> 맥이 인터넷에서 받은 파일에 붙여두는 **"이거 의심스러움" 딱지를 떼는** 명령이에요.
> 이 앱 하나에만 적용되고, 컴퓨터의 다른 부분은 아무것도 건드리지 않습니다.

### 3단계 — 실행!

응용 프로그램 폴더에서 **Premiere Relink**를 더블클릭하세요.
잠깐 기다리면 브라우저가 자동으로 열리면서 사용 화면이 나타납니다.

여기까지 됐으면 설치 끝! 다음부터는 그냥 더블클릭만 하면 됩니다.

## 🎬 사용 방법

1. **앱 실행** — 더블클릭하면 브라우저에 사용 화면이 열립니다.
2. **프로젝트 선택** — 오프라인 클립이 있는 `.prproj` 파일을 고르세요.
3. **검색 폴더 선택** — 영상 파일들이 실제로 들어있는 폴더(외장하드 등)를 지정하세요.
4. **분석** — 앱이 알아서 파일을 찾아 짝을 맞춥니다. 확실한 건 자동으로 연결하고, 헷갈리는 건 후보를 보여주면서 물어봐요.
5. **완료** — 원본 옆에 `프로젝트이름_relinked.prproj` 파일이 새로 생깁니다. **이 파일을 프리미어에서 열면 끝!**

## ❓ 자주 묻는 질문

**Q. 터미널 명령어, 진짜 안전한 거 맞아요?**
네. 저 명령은 "이 앱은 인터넷에서 받은 거라 의심스러움"이라는 딱지 하나를 떼는 것뿐이에요. 파일을 지우거나 설정을 바꾸는 게 아닙니다. 딱 `Premiere Relink.app` 한 개에만 적용돼요.

**Q. 터미널 없이 하는 방법은 없나요?**
있긴 한데 클릭할 곳이 더 많아요. 앱을 더블클릭 → 경고창에서 [완료] → **시스템 설정 → 개인정보 보호 및 보안** 으로 들어가서 맨 아래로 스크롤 → "Premiere Relink이(가) 차단되어 사용할 수 없었습니다" 옆의 **[그래도 열기]** 클릭 → [열기] → 맥 비밀번호 입력. 이 과정이 번거로워서 터미널 한 줄을 추천하는 거예요.

**Q. 2단계에서 "No such file or directory" 라고 떠요.**
앱이 아직 응용 프로그램 폴더에 안 들어가 있다는 뜻이에요. 1단계(앱을 응용 프로그램 폴더로 드래그)를 먼저 했는지 확인하고 다시 해보세요.

**Q. 원본 프로젝트가 망가질 수도 있나요?**
아니요. 원본 `.prproj`는 읽기만 하고 절대 수정하지 않아요. 결과는 항상 별도 파일(`_relinked.prproj`)로 저장되고, 백업까지 자동으로 만듭니다.

**Q. 어떤 맥에서 되나요? 윈도우는요?**
인텔 맥, M칩(Apple Silicon) 맥 **모두** 됩니다. 윈도우 버전은 준비 중이에요.
