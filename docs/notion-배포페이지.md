# Premiere Relink — 프리미어 오프라인 미디어 자동 재연결

> 프리미어 프로에서 **"미디어 오프라인"** 으로 빨갛게 뜨는 클립들을,
> 파일을 하나하나 찾아 클릭하지 않아도 **자동으로 다시 연결**해주는 무료 맥 앱입니다.

외장하드 이름이 바뀌었거나, 폴더를 옮겼거나, 다른 맥으로 프로젝트를 가져왔을 때 —
프로젝트 파일(.prproj)과 영상이 들어있는 폴더만 지정하면 알아서 찾아 연결합니다.

- ✅ **원본은 절대 건드리지 않습니다.** 결과는 항상 새 파일(`_relinked.prproj`)로 저장되고, 작업 전 백업도 자동 생성됩니다.
- ✅ 인텔 맥·M칩(Apple Silicon) 맥 **모두 지원**
- 현재 버전: **v0.1** (베타 — 버그 제보 환영!)

## 🔧 설치 방법 — 처음 한 번만 3단계

> 💬 시작 전에: 실행할 때 "확인할 수 없습니다" 경고가 뜨는데 **정상입니다.**
> 바이러스가 아니라, 제작자가 아직 애플 유료 인증(연 13만원)을 받지 않아서 뜨는 안내예요.
> 아래 3단계만 하면 다음부터는 그냥 더블클릭으로 열립니다.

### STEP 1 — 앱을 응용 프로그램 폴더에 넣기

다운로드한 `Premiere Relink.zip`을 더블클릭해 압축을 풉니다.
안에 있는 **Premiere Relink** 앱을 **응용 프로그램(Applications)** 폴더로 드래그하세요.

### STEP 2 — 더블클릭 → [완료] 누르기

앱을 더블클릭하면 아래 경고가 뜹니다. **여기서는 [완료]만 누르고 닫으세요.**
(지금은 안 열리는 게 정상입니다. 바로 STEP 3으로 넘어가세요.)

> [📷 스크린샷 2 — `dist-assets/screenshots/step2-경고창.png` 삽입]

### STEP 3 — 시스템 설정에서 "그래도 열기" 한 번

1. 화면 왼쪽 위 **애플 로고()** → **시스템 설정**
2. 왼쪽 목록에서 **개인정보 보호 및 보안** 클릭

> [📷 `dist-assets/screenshots/step3-설정진입.png` 삽입 — 개인정보 보호 및 보안 화면]

3. 오른쪽 화면을 **아래로 쭉 스크롤**해서 **"보안"** 부분까지 내려가면 이런 문장이 보입니다:
   > "Premiere Relink"이(가) 차단되어 사용할 수 없었습니다.
4. 그 오른쪽의 **[그래도 열기]** 버튼 클릭

> [📷 `dist-assets/screenshots/step3-보안섹션.png` 삽입 — 아래 사진의 **"다음에서 응용 프로그램 허용" 바로 위**에 차단 문구와 [그래도 열기] 버튼이 나타납니다]

5. 확인 창이 한 번 더 뜨면 **[열기]** 클릭 → 맥 비밀번호(또는 지문) 입력

### ✅ 끝!

이제부터는 그냥 **더블클릭**만으로 실행됩니다.

## 🎬 사용 방법

1. 앱을 더블클릭하면 브라우저가 자동으로 열립니다.
2. **프로젝트 선택** — 오프라인 미디어가 있는 `.prproj` 파일을 선택
3. **검색 폴더 선택** — 영상 파일들이 실제로 들어있는 폴더(외장하드 등)를 지정
4. **분석** — 자동으로 파일을 찾아 매칭합니다. 확실한 건 자동 연결, 애매한 건 후보를 보여주고 직접 고르게 합니다.
5. **완료** — 원본 옆에 `프로젝트명_relinked.prproj` 가 새로 생깁니다. 이 파일을 프리미어에서 열면 끝!

## ❓ 자주 묻는 질문

**Q. 왜 "확인할 수 없습니다" 경고가 뜨나요? 안전한가요?**
애플 개발자 인증(연간 비용 발생)을 아직 받지 않아서 뜨는 경고입니다.
원본 프로젝트와 영상 파일은 **절대 덮어쓰지 않고**, 재연결 결과는 항상 새 파일(`_relinked.prproj`)로 저장하며 작업 전 백업을 만듭니다.

**Q. STEP 3이 헷갈려요. 더 빠른 방법 없나요?**
컴퓨터가 익숙한 분은 터미널이 제일 빠릅니다.
`응용 프로그램 → 유틸리티 → 터미널`을 열고 아래 한 줄을 붙여넣은 뒤 Enter:
```
xattr -cr "/Applications/Premiere Relink.app"
```
그 다음 앱을 더블클릭하면 바로 열립니다.

**Q. 어떤 맥에서 되나요? Windows는요?**
**인텔 맥과 M칩(Apple Silicon) 맥 모두** 지원합니다. Windows 버전은 준비 중입니다.

**Q. 원본 프로젝트가 망가질 수 있나요?**
아니요. 원본 `.prproj`는 읽기만 하고 절대 수정하지 않습니다. 결과는 항상 별도의 `_relinked.prproj` 파일로 저장됩니다.
