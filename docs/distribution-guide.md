# Premiere Relink 설치 방법

> 배포 페이지(노션 등)에 그대로 복사해서 쓰는 안내문. macOS Sequoia(15) 기준.

---

## 설치 방법

### 1. 다운로드 후 압축 풀기
`Premiere Relink.zip`을 다운로드하고 더블클릭해서 압축을 풉니다.
`Premiere Relink.app`을 **Applications(응용 프로그램) 폴더**로 옮겨주세요.

> [스크린샷: Finder에서 Premiere Relink.app을 Applications 폴더로 드래그하는 모습]

### 2. 실행하기
`Premiere Relink.app`을 더블클릭합니다.

처음 실행하면 아래와 같은 경고가 뜹니다. **정상입니다** — 저희가 아직 Apple 개발자 인증을 받지 않아서 뜨는 경고이고, 악성코드가 아닙니다.

> "Premiere Relink"을(를) 열 수 없습니다. Apple에서 악성 소프트웨어가 있는지 확인할 수 없습니다.

> [스크린샷: Gatekeeper 경고 다이얼로그]

이 창에서는 **"완료"만 누르고 닫아주세요.** (다시 시도해도 안 열립니다 — 아래 3번으로 넘어가세요)

### 3. 시스템 설정에서 허용하기
1. 애플 메뉴( ) → **시스템 설정** 클릭
2. 왼쪽 메뉴에서 **개인정보 보호 및 보안** 클릭
3. 화면을 아래로 스크롤하면 다음과 같은 문구가 보입니다:
   > "Premiere Relink"이(가) 차단되어 사용할 수 없었습니다.
4. 옆에 있는 **"그래도 열기"** 버튼 클릭

> [스크린샷: 개인정보 보호 및 보안 화면, "그래도 열기" 버튼 위치 강조]

5. 확인 창이 한 번 더 뜨면 **"열기"** 클릭
6. Mac 로그인 암호(또는 지문/Face ID) 입력

> [스크린샷: 최종 확인 다이얼로그]

### 4. 완료
이제부터는 더블클릭만으로 정상적으로 실행됩니다. 자동으로 브라우저가 열리면서 사용 화면이 나타납니다.

---

## 자주 묻는 질문

**Q. 계속 "손상되었습니다"라고 뜨고 안 열려요.**
A. 터미널(응용 프로그램 → 유틸리티 → 터미널)을 열고 아래 명령어를 붙여넣은 뒤 Enter를 누르세요.
```
xattr -cr "/Applications/Premiere Relink.app"
```
그 다음 다시 더블클릭하면 열립니다.

**Q. 왜 이런 경고가 뜨나요? 진짜 안전한가요?**
A. Apple 개발자 인증(연간 비용 발생)을 아직 받지 않아서 뜨는 경고입니다. 소스 코드는 [여기서/요청 시] 확인 가능하며, 정식 인증을 받으면 이 과정 없이 바로 실행됩니다.

**Q. Windows에서도 되나요?**
A. 아직 macOS 전용입니다. 윈도우 버전은 준비 중입니다.
